# Демонстрационный экзамен


Команда `csu` создаёт администратора с `is_staff=True` (логин и пароль из `.env`, по умолчанию `Admin26` / `Demo20`).

## Запуск

```bash
python manage.py runserver
```

Сайт: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## Основные URL

| Адрес | Описание |
|-------|----------|
| `/` | Личный кабинет — история заявок |
| `/create/` | Оформление заявки |
| `/auth/login/` | Вход |
| `/auth/register/` | Регистрация |
| `/panel/` | Панель администратора (`is_staff=True`) |
