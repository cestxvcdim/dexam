# Демонстрационный экзамен

---
Команда `csu` создаёт суперпользователя с `is_staff=True` (доступ к панели администратора), если пользователь с таким `SUPERUSER_USERNAME` ещё не существует.

---

## Запуск

```bash
python manage.py runserver
```

Сайт будет доступен по адресу: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)

## Основные URL

| Адрес | Описание |
|-------|----------|
| `/` | Список карточек текущего пользователя |
| `/create/` | Создание карточки |
| `/auth/login/` | Вход |
| `/auth/register/` | Регистрация |
| `/panel/` | Панель администратора (только `is_staff=True`) |
| `/admin/` | Стандартная админка Django |
